         New Unlock Code:  FCN + 6-digit security code + new 3-digit
                           Unlock Code + #
           System Select:  RCL/STO + repeatedly press *
                           (One of the following prompts will appear)
                           STD AB or STD BA    (Standard)
                           SCAN BA or SCAN AB  (Alternate)

                           SCAN B
