NEC P-200 & P-300 Programming

The P200( and 300)'s ESN and NAM can be programmed using the P3.exe program. However, a front end program was designed to format the ESN. Using this program requies building a cable that runs between the parallel port of a PC andthe P200's data port. After changing the ESN, the program leaves the phone in test mode, from which you can program the NAM.
