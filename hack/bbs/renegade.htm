Hacking Renegade BBS Systems

by Redwoods of Group 42

Renegade is one of the more common BBS systems being used. Renegade is a hack of Telegaurd, and in the conversion the authors left a bug that can allow you access to the renegade data files, user data files, and the chat logs. All you need is a Renegade Mock Setup and to follow these steps:
  • Dial your target Renegade BSS.
  • Logon. It is advised that you use a fake account and deleting the logs when you're done.
  • After you logon, make sure the sysop is not around. Check the messages, play some door games to make it look normal. Page the sysop to see if he's around. The best time to do this is very late a night when the sysop is asleep.
  • When ready, goto the file menu.
  • In the file menu there is an option called /A. This is the archive menu. This is where you will do your work on compressed files and where you will do your hacking.
  • From the archive menu select W to work on a file.
  • Enter the name of the archive you wish to work on. This can be any name followed by the extension of the file. .zip is the best as most systems support it. What you will be doing is adding the data files to a .zip file for later download.
  • Now you have to add your files. You hit A to do this. When it prompts you for files type ../../u*.dat. Next type ../../r*.dat. And finally type ../../*.log.
  • Now you have the data files and the chat logs in the archive. You may now download the file which you have created.
  • Once you have the files you need to set up a mock Renegade BBS to view the files.
  • Copy the renegade.dat to your main renegade directory and copy the user files to x:\renegade\data, or whatever your renegade data directory is.
  • Start renegade using renegade /l to startup renegade in its local mode.
  • Goto the user setup and start writing down passwords. System passwords are found in the system config section. You can logon as the sysop, but to do so you will need his phone number, passowrd and the system password. You can loook at the renegade.dat file and find pards of passwords as well.
