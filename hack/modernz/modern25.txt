INTERNATIONAL DIRECT CALLS:

011 + country code + city code + telephone number

possible delay after dialing the number before it reaches the country.

busy immediately after dialing generally indicates busy circuits to the

country being calls.

note if a three second delay or more between the 0 and the 1 occurs, you

will reach the local operator.

<><><><><><><><><><><><><><><><><><><><><><><><><><><><><><><><><><><><><><><>
                                                                                                                                                                                                                                                                                                                                                                                                                                                    
OPERATOR ASSITED CALLS:
                                                                                                                                                                                                                                                                                                                                         
01 + country code + city code + telephone number

these calls include collect, bill to third, calling card, time and charges,

and person-to-person calls.

Note that collect and calling card charges to the country being calls may be

restricted.

ASSISTANCE:
                                                                                                                                                                                                                                                                                                                                            
If you definetly need an operator, to find the number or the country code or

for some other reason.

00 with reach the overseas services 

Should you need to find a number overseas.

Overseas directory assistance is currently only provided by AT&T, which can

be reached by dialing 10ATT (thats 10288 on the phone)

Before December 22, 1991 there was no charge for this service... 

Now the charge is $1.50 per number given.

This rate is subject to change.  AT&T is attempting to make it a $3.00 charge

per number given, but the FCC is currently fighting it.
