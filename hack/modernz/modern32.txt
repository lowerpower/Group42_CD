    gateway                                                                    
    =======

    A special application program that allows two different networks       
    (private to public, or TYMNET to non-TYMNET) to communicate and        
    transfer data. Three types of gateways are TYM2, which connects        
    networks that are both using TYMNET packet technology; X.25, which     
    connects a non-TYMNET private network to the TYMNET network; and X.75, 
    which connects an international Public Data Network (PDN) to the       
    TYMNET network.

    Gateway Operations Manager (GOM)                                           
    ================================

    A program that provides the status of a gateway. A set of routines     
    that allows users to determine internal status for its different       
    components, such as port, line, and node management; statistical       
    analysis for performance, errors, and throughput.

    Good Guy list                                                              
    =============

    Assembler/Debugger (NAD) and used by the Node Code and the Kernel code 
    to define access to the trouble- shooting tools DDT and XRAY. The Good 
    Guy list contains usernames, slot access, and DDT and XRAY licenses.   
    GOSIP Government Open Systems Interconnection Profile.
