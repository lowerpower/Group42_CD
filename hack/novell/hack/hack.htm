HACK

The Hack program will grant supervisor equivalency to all users on a network if it is run while the supervisor is logged in. It works (I've tested it). Patches are available from Novell to circumvent this hole. It was discovered by students and professors at Lieden University in the Netherlands.

View is a utility to show users logged in with Supervisor equivalency.

There is a version of hack that will work on a patched Novell server. Novell's patch only hashed the first several bytes of the packet to make it secure, the new hack uses this weakness to get by the patch.
