SA8550 Diagnostic Codez

by G-Man of Group 42

Code F1

    Type - Timeout counter exceeded maximum count.
    Cause - Cable disconnected or no FSK data carrier.
    Display - NC after any keyboard entry.
    Tuning - Authorized barker channel only.
    Action - Refresh converter to restore service. Send security transaction to clear F1 code.

Code F2

    Type - Powerup counter exceeded maximum count.
    Cause - Power disconnected too many times.
    Display - Normal.
    Tuning - Authorized barker channel only.
    Action - Refresh converter to restore service. Send security transaction to clear F2 code.

Code F7

    Type - Phase locked loop tampering.
    Cause - Phase locked loop tampering has been detected. Invalid tuning parameters loaded in NVM.
    Display - Normal
    Tuning - Dependant on tamper condition.
    Action - Look for signs of physical tampering. F7 code will stop when tampering stops.

Code F8

    Type - Disconnect.
    Cause - Converter turned off by disconnect transaction.
    Display - Normal.
    Tuning - Disconnect barker only.
    Action - Refresh converter to restore service. F8 code is cleared automatically.
