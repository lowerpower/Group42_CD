SA8580 Test Chip Code, 6 and 7 Button Versions

by G-Man of Group 42

If you have a 8580 7 button unit, program a MC68HC705C8P with this code (also in intel format.) Remove original 40 pin processor and replace with your newly programmed processor. Your lock out code is 24124.

If you have an 8580 6 button, find the chip with a Green or Red strip on it. This is a serial EEprom, a 93c46. Reprogram it with either the Red or Green code and replace in unit.

It is possible to write some pic code to program the 93c46 serial EEprom with the above .bin files and use this as a test chip. Infact that is how most quick boards work today.
