[8lgm]-Advisory-11.UNIX.sadc.07-Jan-1992

PROGRAM:

	sadc(1M)	(/usr/lib/sa/sadc)
			Supplied with SYS V accounting utilities

KNOWN VULNERABLE OS's:

  • SCO UNIX 3.2v4.2 (suid root)
  • SVR4/i386 4.0.3 (sgid sys)
  • A/UX 2.0.1 (sgid sys)

DESCRIPTION:

sadc(1M) can be used to create files in normally unwritable directories. sadc normally runs egid sys, and therefore can be used to create files in group sys writeable directories.

SCOs sadc runs euid root, and therefore can be used to create files anywhere on the filesystem.

IMPACT:

Any user with access to sadc under SCO can become root. On other vulnerable systems, users may have improved chances of gaining root access.

REPEAT BY:

Exploit details will not be made available, until patches have been provided.

FIX:

Contact your vendor for a fix.

WORKAROUND:

In the meantime, limit access to sadc by changing mode on /usr/lib/sa.
