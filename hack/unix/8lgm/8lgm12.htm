[8lgm]-Advisory-12.UNIX.suid_exec.27-Jul-1991

PROGRAM:

	suid_exec	(Provided with ksh)

VERSION:

11/16/88a

DESCRIPTION:

suid_exec can be used to execute arbitrary programs as root.

IMPACT:

Any user with access to suid_exec can become root.

REPEAT BY:

Exploit details will not be made available at this time.

FIX:

Obtain patch from your vendor.

WORKAROUND:

Remove set bit from suid_exec in the interim.

DISCUSSION:

suid_exec checks permissions on files in a poor manner, and does not verify the interpreter used in a secure fashion.
