Welcome to Group 42's Identification Archive. This is a guide to help you identify the validity of identification cards used in North America.

From the main ID page (of either the USA or Canada) select the state or provence you wish to view ID cards from. This will bring up a list of actual size graphic representations of the ID cards along the left hand side of the page. On the right hand side of the page are notes pertaining to the ID shown. These notes are accurate to the best of our knowledge and can describe ID that is a slight variant to those shown.

By clicking on the actual size graphic of the ID card a higher resoultion graphic of the ID card will be shown. This graphic is exactly two times the original size of the ID card. The oversized graphic is needed to show detail of the ID card on standard video displays. Printing these graphics out at 150 DPI will show the correct ID card size.
