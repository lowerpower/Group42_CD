Cool Edit Crack

In your COOL.INI file do the following :

[Registration]
User=any name you want
Number=ANSERME


Next time you fire up the program it will tell you what your serial number should be for your name that you typed in. Write this number down an replace 'ANSERME' in the COOL.INI file. You will now have a registered copy in your (or any) name.
